Frequently Asked Questions

Who needs this tool?
Marketers mostly, so they can look which competitors are mentioned by AI, in which order, what are the resources and websites mentioned by AI, etc.
How much does it cost?
The tool is free and open-source. You only need to pay for using AI models via selected provider (like OpenAI, Claude or OpenRouter).
How secure is it?
It runs on your computer. AICW sends questions to selected AI models via selected AI model provider - check Terms of Use and Privacy Policy of these providers for more information.
I want to add a feature or report an issue?
AICW tool is open-source, check the source code at GitHub and add a feature or report an issue.
